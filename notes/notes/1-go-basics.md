## 核心概念

### 1. 变量声明
```go
var a int = 10    // 显式
b := 20           // 推断
const pi = 3.14   // 常量
```

### 2. 数组 vs 切片
| 特性   | 数组       | 切片         |
| ---- | -------- | ---------- |
| 长度   | 固定       | 动态         |
| 声明   | `[5]int` | `[]int`    |
| 追加   | 不可       | `append()` |
| 使用场景 | 极少       | **绝大多数情况** |

### 3. map
```go
m := make(map[string]int)
m["key"] = value
v, ok := m["key"]  // 取值+判断存在
delete(m, "key")
```

### 4. 多返回值
```go
func calc(x, y int) (int, int) {
    return x+y, x-y
}
sum, diff := calc(10, 3)
```

## 与Python对比
| Python | Go      |
| ------ | ------- |
| `list` | `slice` |
| `dict` | `map`   |
| 无多返回值  | 原生支持    |
| 动态类型   | 静态类型    |

## 关键
- 切片是Go核心，替代数组使用
- := 只能在函数内使用
- 多返回值是Go特色，错误处理常用
